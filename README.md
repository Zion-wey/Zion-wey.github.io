## ⚠️ 重要：添加作品图片

网页已配置好灯箱画廊系统，你只需将**图片文件**放入对应文件夹即可自动展示。

### 📂 文件放置指南

请把你的作品图片（jpg/png 格式）按照以下路径放置：

```
images/
├── avatar.jpg                          ← 你的头像
└── portfolio/
    ├── plush-toy/                      ← 🧸 毛绒玩具创意设计比赛（6张）
    │   ├── 01.jpg                      ← 第1张设计展板
    │   ├── 02.jpg                      ← 第2张设计展板
    │   ├── 03.jpg                      ← 第3张设计展板
    │   ├── 04.jpg                      ← 第4张设计展板
    │   ├── 05.jpg                      ← 第5张设计展板
    │   └── 06.jpg                      ← 第6张设计展板
    ├── posters/                        ← 🎨 海报设计
    │   ├── welcome-01.jpg              ← 校园迎新晚会海报
    │   ├── club-01.jpg                 ← 校园社团招新海报
    │   └── art-01.jpg                  ← 校园文化艺术节海报
    ├── promo/                          ← 📋 宣传物料
    │   ├── banner-01.jpg               ← 校园活动易拉宝设计
    │   └── kit-01.jpg                  ← 校园迎新宣传物料包
    └── video/                          ← 🎬 摄影影像
        ├── antifraud-01.jpg            ← 《守住钱袋子》反诈短片
        └── event-01.jpg                ← 校园迎新晚会活动记录
```

### 操作步骤

1. 打开文件夹：`C:\Users\17991\personal-website\images\portfolio\`
2. 把 **前 6 张作品图** 放入 `plush-toy\` 文件夹，重命名为 `01.jpg` ~ `06.jpg`
3. 把其余作品图放入对应的文件夹（posters / promo / video）
4. 把头像放入 `images\` 文件夹，重命名为 `avatar.jpg`
5. 刷新浏览器即可看到效果

> 💡 支持 jpg 和 png 格式。建议图片宽度不超过 2000px，以保证加载速度。
> 如果某个作品只有 1 张图，灯箱会直接展示，不显示缩略图导航。
> 如果有 2 张及以上，灯箱会显示缩略图条和左右翻页按钮。

---

## 🖼️ 如果需要添加更多作品

在 `index.html` 中搜索 `portfolioItems`，按照已有格式添加新的作品条目（包含图片路径即可）。

## 🎨 自定义颜色

在 `index.html` 开头的 `:root` 中修改 CSS 变量（推荐用取色器选择喜欢的浅蓝色调）。
